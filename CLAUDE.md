# Project Overview

This is a data science project investigating quantitative proteomics data obtained from cohorts of patients with mental health disorders and healthy controls.

# Work Style

I will be using .py notebooks instead of .ipynb notebooks for this project. Do not run any python or R commands, I will run the notebooks and scripts myself.

# Directory Restrictions

- `data/`: Raw and processed data. No read, edit, or write access.
- `src/`: Third-party scripts and applications. Read-only; no edit or write access.
